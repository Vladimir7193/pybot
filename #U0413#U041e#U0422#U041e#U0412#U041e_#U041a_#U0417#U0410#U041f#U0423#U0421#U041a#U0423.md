## НАСТРОЙКА

### Если сигналов слишком мало:

1. Отключить продвинутые фичи:
```python
USE_FIBONACCI = False
USE_AMD = False
USE_KILL_ZONES = False
```

2. Снизить минимальные пороги:
```python
SETUP_PATTERN_CONFIDENCE_MIN = 0.6  # было 0.7
KEY_LEVEL_IMPORTANCE_MIN = 0.7      # было 0.8
```

3. Использовать RELAXED mode:
```python
SMC_MODE = "RELAXED"
```

---

### Если сигналов слишком много:

1. Включить все продвинутые фичи:
```python
USE_FIBONACCI = True
USE_AMD = True
USE_KILL_ZONES = True
USE_MOMENTUM = True
USE_RANGE_DETECTION = True
```

2. Повысить минимальные пороги:
```python
SETUP_PATTERN_CONFIDENCE_MIN = 0.8  # было 0.7
KEY_LEVEL_IMPORTANCE_MIN = 0.9      # было 0.8
```

3. Сделать некоторые проверки REQUIRED:
```python
ORDERFLOW_SEQUENCE_REQUIRED = True  # было False
KILL_ZONE_REQUIRED = True           # было False
MOMENTUM_REQUIRED = True            # было False
```

## ПОДДЕРЖКА

**Решение:**
1. Смотреть логи: `tail -f logs/rocketbot.log`
2. Посмотреть какие проверки не проходят
3. Попробовать RELAXED mode
4. Отключить продвинутые фичи

**Решение:**

2. Включить продвинутые фичи
3. Повысить минимальные пороги
