## 3️⃣ Режимы сигналов

### 🔥 STRICT MODE
- **Все фильтры** обязательны
- **1-3 сигнала** в неделю
- **Максимальное качество**
- Win Rate: ~70-80%

### ⚡ RELAXED MODE
- **Только ключевые** фильтры
- **5-10 сигналов** в день
- **Хорошее качество**
- Win Rate: ~60-70%

## 4️⃣ Настройки

### Текущие (оптимальные):
```python
# config.py
SYMBOLS = ["BTCUSDT", "ETHUSDT"]
RISK_PER_TRADE = 0.02  # 2%
MAX_POSITIONS = 2

# SMC фильтры
ENABLE_SMC = True
USE_SETUP_PATTERNS = False
USE_ORDER_FLOW = False
USE_KEY_LEVELS = False
```

### Для большего количества сигналов:
```python
USE_SETUP_PATTERNS = True
USE_ORDER_FLOW = True
USE_KEY_LEVELS = True
```

## 📞 Поддержка

**Нет сигналов?**
1. Подождите 15 минут (интервал сканирования)
2. Проверьте логи: `logs/rocketbot.log`
3. Убедитесь, что Telegram настроен в `.env`

**Хотите больше сигналов?**
1. Включите фильтры в `config.py`
2. Перезапустите бота
